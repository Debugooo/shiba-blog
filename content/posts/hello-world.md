# 欢迎来到我的博客！

这是我使用 Next.js 搭建的第一个博客系统。让我来介绍一下它的主要功能。

## ✨ 主要特性

### 1. Markdown 支持

你可以使用 Markdown 语法来编写文章，包括：

- **粗体** 和 *斜体*
- 列表（有序和无序）
- 代码块
- 引用

### 2. 代码高亮

支持代码块的显示：

```javascript
function hello() {
  console.log('Hello, World!')
}
```

### 3. 暗色模式

点击右上角的按钮可以切换亮色/暗色主题，保护你的眼睛。

### 4. RSS 订阅

支持 RSS 订阅功能，你可以通过 RSS 阅读器订阅我的博客，第一时间获取最新文章。

## 🚀 技术栈

这个博客系统使用了以下技术：

- **Next.js 14** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Markdown** - 文章编写

## 📝 开始写作

在 `content/posts` 目录下创建 `.md` 文件即可开始写作。记得在文件头部添加 frontmatter：

```yaml
---
title: 文章标题
date: 2024-04-07
excerpt: 文章摘要
---
```

希望你喜欢这个博客系统！
